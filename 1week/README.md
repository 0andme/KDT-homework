+ HTML
설계한 것을 보며 HTML을 아주 간단하게 작성하였다. header 태그가 이미 존재하지만 c와 c++을 자주 사용했어서 div태그와 class속성으로 코드를 작성하는걸 개인적으로 더 선호하는 편이다.~~아직은 많이 몰라서 이기도 하다.~~
